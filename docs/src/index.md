# Conplete.jl

**Conplete** is a library for Julia janguage implementing polynomial reductions for several NP-Complete problems. It utilizes interfaces described in chapter [interfaces](interfaces.md)

Full list of structures describing problem instances implemented in library can be found in chapter [structures](structures.md).

Library allows for extending its functionality by the user, which is described in chapter [Extending library](extending.md)

Library was created for a master's thesis of Paweł Stanik in Wrocław university of science and technology.
